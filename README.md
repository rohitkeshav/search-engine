# Search Engine
  Compressed trie based implementation of a search engine

## Getting Started

 Run the following in python 3 environment - 
 
 Environment Setup - 
 
    pip install requirements
 
 Searching for query - 
 
    python app.py -q "<search query>"

## Example Usage

    python app.py -q "What is Science?"
